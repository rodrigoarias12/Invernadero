Proyecto Invernadero IOT  
Es un modelo a escala de un invernadero con riego y control de temperatura. 
Resumiendo, se utiliza una placa Arduino UNO que interactúa con sensores en tiempo real y actuadores para el control de ambiental. 
La aplicación mobile nos permite comunicar por medio de bluetooth con la placa y ver la configuración de la administración de nuestroinvernadero. 
Dejo el GIT con ambos proyectos (Arduino y Android) y la descripción de todos los elementos físicos tanto sensores como placas y actuadores. 
Documentación del proyecto https://github.com/rodrigoarias12/Invernadero/blob/master/Doc/Informe_Final.pdf
